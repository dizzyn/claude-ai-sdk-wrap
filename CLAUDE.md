# Project: Czech Republic Laws in Construction Proceedings

## Project Description
This project is designed to work with the laws of the Czech Republic in the field of construction proceedings. It contains a collection of laws, case law, regulations, and related documents necessary for navigating construction law.

## Working Directory
All work takes place in the `workspace/` directory.

## Working Rules

1. **Work in the `workspace/` directory is performed exclusively by the Code Agent according to user instructions.**
2. Creating various indexes and auxiliary files is permitted for better orientation and greater understanding of the content.
3. When creating indexes and auxiliary files, the existing directory structure must be maintained.
4. All changes in `workspace/` must be made based on explicit user instruction.
5. **Always respond in the language of the user's query.**
